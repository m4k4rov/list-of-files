**Современный файловый менеджер с экспортом метаданных**

## 🎯 Описание

Modern File Explorer — это современный файловый менеджер на Python с графическим интерфейсом, который позволяет:

- 🔎 Просматривать файлы и папки в выбранной директории (рекурсивно)
- 🧾 Отображать расширенные метаданные файлов:
  - Размер (с автоматическим форматированием в Б/КБ/МБ)
  - Даты создания, изменения и последнего доступа
- ⏱️ Отслеживать прогресс обработки больших директорий
- 💾 Экспортировать список файлов и их метаданные в форматы:
  - Microsoft Word (.docx)
  - Excel (.xlsx)

## ✨ Ключевые особенности

- **🌙 Тёмная тема** — комфортный интерфейс для длительной работы.
- **🚀 Многопоточность** — интерфейс не «зависает» при сканировании больших папок.
- **📊 Индикатор прогресса** — показывает ход обработки файлов в реальном времени.
- **🗂️ Рекурсивный обход** — сканирует все вложенные папки.
- **🛡️ Обработка ошибок** — корректно обрабатывает недоступные папки и файлы.
