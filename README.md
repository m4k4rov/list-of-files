# 📁 Modern File Explorer — приложение для управления файлами

Удобное приложение для переименования файлов и просмотра содержимого папок с возможностью экспорта данных! 💾✨

## 🎯 Основные возможности

* **Переименование файлов** с порядковыми номерами 🔢
* **Сортировка** по имени, дате изменения или создания 🗂️
* **Просмотр содержимого папок** (включая подпапки) с подробной информацией о файлах 👁️
* **Экспорт списка файлов** в DOCX и XLSX 💽
* **Прогресс‑бар** для длительных операций 📊
* **Многопоточность** — интерфейс не зависает при обработке больших папок 🧵
* **Рекурсивный обход** папок с настройкой глубины сканирования 🌀
* **Обработка ошибок** доступа к файлам и папкам 🛡️

## 🖥️ Интерфейс приложения

Приложение имеет две вкладки:

### 1️⃣ Вкладка «Переименование файлов» 🔄

![Переименование](https://via.placeholder.com/600x400?text=Rename+Tab)

**Элементы управления:**

* 📂 **«Папка»** — поле для указания пути к папке с файлами
* 🔍 **«Обзор»** — кнопка для выбора папки через диалоговое окно
* 📝 **«Ключевое слово»** — текст, который будет добавлен в имя каждого файла (например, `report` → `report_001.txt`)
* 🔀 **«Сортировка по»** — выбор критерия сортировки:
    * «Имя файла» — по алфавиту 🆎
    * «Дата изменения» — от старых к новым 🕰️
    * «Дата создания» — от старых к новым 🗓️
* 🔧 **«Переименовать файлы»** — запускает процесс переименования
* 📜 **Область лога** — отображает ход выполнения операций и сообщения об ошибках:
    * Время выполнения 🕒
    * Статус операции ✅/❌
    * Детали переименования (`old_name.txt → new_name_001.txt`)

---

### 2️⃣ Вкладка «Список файлов» 📋

![Список файлов](https://via.placeholder.com/600x400?text=File+List+Tab)

**Элементы управления:**

* 📂 **«Выбрать папку»** — открывает диалоговое окно для выбора папки
* 💾 **«Экспорт в DOCX»** — сохраняет список файлов в формате Word‑документа (со стилями и форматированием)
* 💾 **«Экспорт в XLSX»** — сохраняет список файлов в формате Excel‑таблицы (с автоподбором ширины колонок)
* 📈 **Прогресс‑бар** — показывает ход сканирования папки (в % и количестве обработанных элементов)
* 🌳 **Дерево файлов** — таблица с информацией о файлах и папках:
    * **Имя файла/папки** — относительный путь от выбранной папки 🛤️
    * **Тип** — «Файл» 📄 или «Папка» 📁
    * **Размер** — в Б, КБ или МБ (автоматически подбирается единица измерения) 📏
    * **Создан** — дата и время создания файла/папки 🗓️🕰️
    * **Изменён** — дата и время последнего изменения ✏️🕰️
    * **Открыт** — дата и время последнего доступа 👁️🕰️
* 📊 **Строка статуса** — отображает общую статистику:
    * Общее количество элементов 🔢
    * Количество файлов 📄
    * Количество папок 📁

## 🧰 Как пользоваться

### Для переименования файлов:

1. Перейдите на вкладку **«Переименование файлов»** 🔄.
2. Выберите папку с файлами через кнопку **«Обзор»** 🔍.
3. Укажите **ключевое слово** (опционально) 📝.
4. Выберите **критерий сортировки** 🔀.
5. Нажмите **«Переименовать файлы»** 🔧.
6. Следите за ходом выполнения в **области лога** 📜.

### Для просмотра и экспорта списка файлов:

1. Перейдите на вкладку **«Список файлов»** 📋.
2. Нажмите **«Выбрать папку»** 📂.
3. Дождитесь завершения сканирования (отслеживайте прогресс 📈).
4. Просмотрите список файлов в таблице 🌳.
5. Для экспорта нажмите:
    * **«Экспорт в DOCX»** 💾 — для создания Word‑документа.
    * **«Экспорт в XLSX»** 💾 — для создания Excel‑таблицы.

## ⚠️ Важные замечания

* ⚠️ Приложение **не создаёт резервные копии** файлов перед переименованием. Будьте осторожны!
* ⚠️ При экспорте в XLSX/DOCX **закройте файл**, если он уже открыт в другой программе.
* ⚠️ Для больших папок сканирование может занять время — используйте прогресс‑бар для отслеживания 📈.
* ⚠️ Некоторые системные папки могут быть недоступны из‑за ограничений прав доступа 🔒.

## 🛠️ Технические детали

* **Интерфейс**: CustomTkinter (тёмная тема, синяя цветовая схема) 🎨
* **Языки**: Python 3.x 🐍
* **Библиотеки**:
    * `customtkinter` — GUI 🖼️
    * `os` — работа с файловой системой 🗂️
    * `datetime` — форматирование дат 🗓️
    * `CTkTreeview` — таблица с древовидной структурой 🌳
    * `python-docx` — экспорт в DOCX 💼
    * `openpyxl` — экспорт в XLSX 📊
    * `threading` — многопоточность 🧵
