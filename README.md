# 📁 Приложение для управления файлами

Удобное приложение для работы с файлами: переименования, просмотра и организации! 💼✨

## 🎯 Основные возможности

* **🔄 Переименование файлов** с порядковыми номерами и ключевым словом.
* **📋 Просмотр списка файлов** с подробной информацией (размер, даты создания/изменения и т. д.).
* **🗂️ Организация файлов** по расширениям в отдельные папки.
* **💾 Экспорт списка файлов** в форматы DOCX и XLSX.
* **⏱️ Индикатор прогресса** для длительных операций.

---

## 🧭 Интерфейс приложения

Приложение состоит из **трёх вкладок**:

### 1. 🔄 Переименование файлов

Позволяет массово переименовать файлы в выбранной папке.

**Что можно сделать:**
* 📂 Выбрать папку с файлами.
* ✍️ Задать **ключевое слово** для новых имён (например, `photo` или `document`).
* 🔎 Выбрать **критерий сортировки**:
    * «Имя файла» — по алфавиту;
    * «Дата изменения» — от старых к новым;
    * «Дата создания» — от старых к новым.
* 🔢 Файлы получат имена вида: `{ключевое_слово}_{номер:03d}{расширение}` (например, `photo_001.jpg`, `photo_002.jpg`). Если ключевое слово не указано, будет только номер.
* 🖨️ Запустить переименование кнопкой «Переименовать файлы».
* 📝 Просмотреть **лог операций** в текстовом поле внизу вкладки.

**Пример:**
* Было: `IMG_001.jpg`, `IMG_002.png`, `document.pdf`.
* Ключевое слово: `vacation`.
* Стало: `vacation_001.jpg`, `vacation_002.png`, `vacation_003.pdf`.

### 2. 📋 Список файлов

Отображает все файлы и папки в выбранной директории с их свойствами.

**Элементы управления:**
* 📂 «Выбрать папку» — открывает диалог выбора директории.
* 💾 «Экспорт в DOCX» — сохраняет список в документ Word.
* 💾 «Экспорт в XLSX» — сохраняет список в таблицу Excel.

**Информация о каждом элементе:**
* **Имя файла/папки** (с относительным путём).
* **Тип** (`Файл` или `Папка`).
* **Размер** (в Б, КБ или МБ).
* **Дата создания**.
* **Дата последнего изменения**.
* **Дата последнего доступа**.

**Дополнительно:**
* ⏱️ **Индикатор прогресса** показывает ход сканирования большой папки.
* 📊 **Строка статуса** отображает общее количество найденных элементов.

### 3. 🗂️ Организация файлов по расширениям

Автоматически сортирует файлы из выбранной папки по их расширениям.

**Как это работает:**
1. 📂 Выберите папку для сканирования.
2. 🔎 Нажмите «Сканировать папку» — приложение найдёт все файлы (включая подпапки) и покажет их список.
3. 🗃️ Нажмите «Организовать файлы» — начнётся процесс копирования:
    * Создаётся папка `Organized_Files` внутри выбранной директории.
    * Для каждого расширения (`.jpg`, `.pdf`, `.txt` и т. д.) создаётся отдельная папка.
    * Файлы копируются в соответствующие папки.
    * Если файл с таким именем уже существует, к имени добавляется суффикс `_1`, `_2` и т. д.
4. ⏱️ Ход операции отображается на **индикаторе прогресса**.

**Пример структуры после организации:**
```
Исходная_папка/
├── Organized_Files/
│   ├── jpg/
│   │   ├── photo_1.jpg
│   │   └── photo_2.jpg
│   ├── pdf/
│   │   └── document.pdf
│   └── no_extension/
│       └── file_without_ext
```

## ⚙️ Технические детали

* **Интерфейс:** `customtkinter` (тёмная тема, синяя цветовая схема).
* **Экспорт:**
    * DOCX — через библиотеку `python-docx`.
    * XLSX — через `openpyxl`.
* **Многопоточность:** длительные операции (сканирование, организация) выполняются в фоновом потоке, чтобы не «замораживать» интерфейс.
* **Рекурсивный обход:** все операции работают с папками и их подпапками.

## 💡 Советы по использованию

* Перед массовым переименованием или организацией **сделайте резервную копию** важных файлов.
* Для больших папок используйте индикатор прогресса — это поможет оценить время ожидания.
* При экспорте в XLSX/DOCX убедитесь, что в списке есть данные (вкладка «Список файлов» должна быть заполнена).
* Если операция зависла, перезапустите приложение — фоновые потоки завершатся автоматически.

## ⚠️ Ограничения и известные проблемы


* При переименовании **не проверяется**, существует ли файл с новым именем — возможна перезапись! ⚠️
* Для доступа к системным папкам (например, `C:\Windows`) могут потребоваться **права администратора**.
* Очень большие папки (тысячи файлов) могут обрабатываться долго — следите за прогрессом.

## ❓ Помощь и поддержка

Если вы столкнулись с ошибкой:
1. Проверьте, правильно ли выбрана папка.
2. Убедитесь, что у вас есть права на чтение/запись в этой папке.
3. Посмотрите **лог** (вкладка «Переименование») или **строку статуса** (вкладки «Список» и «Организация») — там может быть описание ошибки.
4. Если проблема не решается, перезапустите приложение.
